# Pasture Runner (ARC-001)

**Category:** Cavern-Run  
**Engine:** Endless Runner  
**Lineage:** First ancestor of the Cavern-Run line in the CaveCode Arcade Planet.

Pasture Runner is a lightweight endless runner designed for phones and low-power devices.  
You control a small green runner (cow block) dodging rolling hay bales.  
Survive as long as possible and beat your best time.
