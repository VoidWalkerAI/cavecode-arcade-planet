# Haze Maze

Infinite-style match-3 and grid puzzle generators.

This folder will hold CaveCode specs + HTML games where:
- You swap tiles.
- Match-3 (or more) clears them.
- New tiles fall in.
- Variants: timed, move-limited, endless, special tiles.

Start new games here by copying from `../_templates/cavecode-game-template.txt`.
